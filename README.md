# Classbridge

A modern learning management interface built with React and Tailwind CSS.

## Prerequisites
- Node.js >= 18
- npm

## Getting Started

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Run tests:
```bash
npm test
```

Build for production:
```bash
npm run build
```
